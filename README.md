# advent-25-clj

Learning some Clojure by solving Advent of Code's 2025 problems

## Running the solutions

To run any of the solutions, you need `clojure` and `lein` installed.
Use `lein run` and the name of the file you want to run. For example:

```sh
lein run day-1
```

*Note:* The file should have a `-main` function for this `lein` script to work.
